# Learning

학습 목표, 현재 이해, 오개념, 재현 과제와 후속 확인을 둔다.

## 학습 구분

- `필수`: 현재 결정, 검증, 위험 또는 복구를 위해 지금 알아야 하는 내용
- `전략적`: 여러 작업에서 반복되거나 다른 분야에도 전이되는 기반 지식
- `호기심`: 당장 필요하지 않아도 사용자가 시간이 있을 때 깊게 탐구하고 싶은 내용

## 승격 규칙

- 먼저 Run의 `후속 학습 후보`에 남긴다.
- 사용자가 학습을 선택하거나 비슷한 개념이 반복해서 필요해질 때 별도 학습 노트로 승격한다.
- 후보와 추론은 검증된 지식으로 취급하지 않는다.
- 원시 대화 전체를 복사하지 않고 학습 목표, 현재 이해, 근거, 재현 과제와 후속 확인만 정리한다.
