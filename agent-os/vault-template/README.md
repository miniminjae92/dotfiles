# Developer OS

개발 프로젝트, 기술 학습, 서비스 설계, 생산성 개선과 관련 통찰을 관리하는 개인 지식 저장소다.

## 원칙

- 빠른 기록은 `00 Inbox`에 둔다.
- 프로젝트 원본 코드는 각 Git 저장소에 둔다.
- 사용자 생각과 에이전트 추론의 출처를 구분한다.
- 검증된 내용만 `20 Knowledge`로 승격한다.
- 원시 Codex 세션과 비밀정보는 저장하지 않는다.
- `_System/Bases`에서 Inbox, Runs, Knowledge, Reviews를 조회한다.
- 로컬 Git 스냅샷은 복구 이력이며 오프디바이스 백업을 대체하지 않는다.
