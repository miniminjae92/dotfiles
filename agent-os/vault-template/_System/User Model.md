# User Model

## 확정된 가치와 선호

- 어려운 문제와 시행착오를 피하지 않는다.
- 복구 가능한 환경에서 에이전트의 자율성을 최대화한다.
- 핵심 결정은 원리, 대안, 위험, 검증 방법까지 이해한다.
- 실행은 에이전트에 적극적으로 위임하고, 사용자는 목적·결정·검증·위험·복구에 집중한다.
- 시간이 허용되면 당장의 필수 이해를 넘어 반복 가치가 높은 기반 지식과 호기심이 생긴 주제를 더 깊게 공부한다.
- 반복되는 수동 불편은 기록하고 자동화 후보로 만든다.
- 개발 실행과 학습이 함께 향상되어야 한다.

## 추론 후보

에이전트가 추론한 성향은 이 영역에 후보로 기록하고 사용자 승인 전에는 확정 사실로 사용하지 않는다.

## 금지

- 비밀정보와 민감한 개인정보를 저장하지 않는다.
- 일시적인 감정이나 단일 대화의 표현을 영구 성향으로 일반화하지 않는다.
- 회사 비공개 정보, 인증정보, 건강·재정 정보는 저장하지 않는다.
- 생산성을 높이는 데 필요하지 않은 사생활 세부사항은 저장하지 않는다.

## 사용 규칙

- 현재 작업과 관련된 확정 선호만 컨텍스트에 넣는다.
- 추론 후보는 의사결정 근거로 단독 사용하지 않는다.
- 사용자가 정정하면 기존 문장을 덮어쓰기보다 변경 이유를 남긴다.
