# Term — 유비쿼터스 랭귀지(UL) 형성 채널

`term`의 목적은 단순 사전 조회가 아니라 **유비쿼터스 랭귀지 형성**이다.
세션·에이전트(Claude·Codex·Gemini)마다 같은 개념을 다른 말로 불러 생기는 불일치를 없앤다.
설명이 곧 협상이고, 사용자가 끄덕인 순간 그 말이 **표준어(합의된 용어)**가 되어 이후 모든 세션을 구속한다.
목적은 여전히 "포착 마찰 0" — 답은 짧고 즉각적이며 본 작업을 끊지 않는다.

## 입력

`/term <용어들>` / `$term <용어들>` 뒤에 명사·구를 공백/쉼표로 나열. 한/영/혼용 허용.
- 정확한 명사를 몰라도 짧은 구로 됨: `/term 왜 캐시를 무효화한다는거`
- 용어 없이 `/term`만 → 직전 대화에서 모를 법한 용어 1~3개를 골라 물어본 뒤 정리.

## 응답 규칙 (즉시 · 초간결)

각 용어를 아래 압축 블록으로, 서론·맺음말 없이 바로 나열한다.

```
**<표준어>** — <이 세션 맥락에서의 뜻 한 줄>
  예) <이 대화/코드에 맞는 구체 예시 한 줄>
  ⚠ <혼동하기 쉬운 X와의 차이 한 줄>
  ↳ 표준어로 굳힐까? (지양할 동의어: <A, B ...>)   ← 합의를 유도하는 한 줄
```

- 정의는 사전적 뜻이 아니라 **"지금 여기서 왜 이 말이 나왔나"**를 먼저 잡는다.
- 확실치 않으면 추측 대신 "이 맥락상 아마 X — 확인 필요"로 표시한다.
- 동의어가 흩어져 있으면 그중 하나를 표준어로 제안하고 나머지는 "지양"으로 묶는다.

## 합의 → 표준어

사용자가 긍정하거나 이미 그 뜻에 동의한 상태면 **상태: 합의**로 기록한다.
확답이 없으면 **상태: 초안**으로 저장하고, 초안은 강제하지 않는다(설명만 한 상태).
합의된 표준어만 이후 세션에서 채택·매핑의 대상이 된다.

## 저장 — 하이브리드 (컨텍스트별)

- **공통어**(개인 어휘, 프로젝트 무관): `/Users/miniminjae/.obsidian/mimir/20 Knowledge/용어사전.md`
- **도메인 특수어**(특정 프로젝트 안에서만 통하는 뜻): 그 repo의 `docs/ubiquitous-language.md`
  (없으면 만들고, repo의 `AGENTS.md`/`CLAUDE.md`에 정본 포인터 한 줄이 없으면 추가를 제안한다.)
- 현재 위치가 프로젝트 repo가 아니거나 용어가 일반어면 공통어로 간다. 애매하면 사용자에게 한 번 확인.

항목 형식:
```
## <표준어> {#normalized-key}
- 컨텍스트: 공통 | <프로젝트명>
- 정의: <한 줄>
- 예시: <한 줄>
- 구분: <혼동 대상과의 차이>
- 동의어/지양: <표준어로 매핑할 표현들>   ← 없으면 생략
- 상태: 합의 | 초안
- 기록: <YYYY-MM-DD> · <어느 세션/주제에서 나왔나>
```

규칙:
1. 쓰기 전 같은 컨텍스트에 해당 용어가 있는지 확인(대소문자·공백·한/영 표기 무시).
2. **없으면** append. **있고 뜻이 일치**하면 중복 생성 금지, 필요 시 `- (YYYY-MM-DD) 추가 맥락: …` 한 줄만.
3. **있는데 정의가 충돌**하면 조용히 덮지 말고 충돌을 드러내 조율한 뒤 갱신한다.
4. 여러 용어는 새 항목만 모아 한 번에 append. 날짜는 세션의 현재 날짜.
5. 파일이 없으면 상단 헤더(설명 한 줄)와 함께 새로 만든다. 관련 노트는 `[[파일명]]`으로 링크.

## 이후 세션에서의 구속 (일관성 강제 = "필요할 때만 지적")

합의된 표준어가 있으면:
- **채택**: 에이전트는 자기 출력에서 표준어를 쓰고, 사용자의 동의어는 **조용히** 표준어로 매핑해 이해한다.
- **지적은 필요할 때만**: 사용자나 에이전트가 비표준어를 써서 **혼동이 생기거나 합의된 정의와 어긋날 때만**
  마지막에 한 줄로 `📎 사전: 여기선 'X'로 부르기로 함 (← Y)`. 무해한 동의어는 지적하지 않는다.
- 새 세션 시작 시나 특정 프로젝트에 들어갈 때, 해당 컨텍스트의 UL 파일을 참고해 표준어를 맞춘다.

## 탈선 금지

이 스킬 호출은 곁가지 채널이다. 용어 정리·기록 외에 진행 중이던 계획·결정·코드를 이 턴에서 건드리지 않는다.
직전에 작업이 있었다면 블록 뒤 마지막 한 줄로만 `↩︎ 원래 작업: <한 줄>`.

한 줄 정의로 먼저 짚는다("모르는 줄도 몰랐던" 사각지대용). "끄자"로 중단.
